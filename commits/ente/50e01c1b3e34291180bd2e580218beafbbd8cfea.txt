Add V2 support for videos and live photos

Extends V2 high-quality widget rendering to support videos and live photos:
- Local videos/live photos: Request 1024px thumbnails from PhotoManager
- Remote videos/live photos: Fall back to V1 (512px) to avoid large downloads
- Split _captureFileV2 into specialized methods for each file type
