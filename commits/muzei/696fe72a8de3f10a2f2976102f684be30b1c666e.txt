Rewrite GallerySettingsActivity photo grid in Compose

This commit rewrites the photo grid in `GallerySettingsActivity` from using a `RecyclerView` to using the new `GalleryChosenPhotoGrid` Composable.

This involves:
- Adding a `GalleryChosenPhotoGrid.kt` file with the Composable function.
- Updating `GallerySettingsActivity` to use `ComposeView` and call `GalleryChosenPhotoGrid`.
- Removing the `RecyclerView` specific code, including the `GalleryAdapter` and related layout resources.
- Updating `MultiSelectionController` to use `SnapshotStateSet` for Compose compatibility.
- Adding a `PaddingValuesExt.kt` to allow for combining `PaddingValues`.
- Updating `GalleryChosenPhotoItem` to accept a `Modifier`.
- Removing unused dimension resources.
