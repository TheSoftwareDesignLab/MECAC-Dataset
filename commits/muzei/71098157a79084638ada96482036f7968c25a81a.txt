Bump Coil to 2.7.0, OkHttp to 5.1.0, and Retrofit to 3.0.0

This commit updates several key dependencies:

- Coil is updated from 1.3.2 to 2.7.0. This includes API changes such as replacing `clear()` with `dispose()` for ImageViews and changes to the `Transformation` interface.
- OkHttp is updated from 3.12.12 to 5.1.0. This involves minor API adjustments like accessing response body and code via properties.
- Retrofit is updated from 2.9.0 to 3.0.0.
