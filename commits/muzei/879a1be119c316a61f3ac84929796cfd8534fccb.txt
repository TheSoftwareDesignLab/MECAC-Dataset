Refactor: Move Gallery settings classes to settings package

This commit moves all UI-related classes for the Gallery source's settings screen into a new `com.google.android.apps.muzei.gallery.settings` package.

The following files have been moved and their package declarations updated:
- `GallerySettingsActivity.kt`
- `GalleryAdd.kt`
- `GalleryChosenPhotoGrid.kt`
- `GalleryChosenPhotoItem.kt`
- `GalleryEmpty.kt`
- `GalleryEmptyStateGraphic.kt`
- `GalleryImportPhotosDialogFragment.kt`
- `GallerySelectionToolbar.kt`
- `GallerySettingsViewModel.kt`
- `GalleryTopAppBar.kt`
- `MultiSelectionController.kt` (moved from `com.google.android.apps.muzei.util`)

This refactoring improves code organization by grouping related settings components together. The `AndroidManifest.xml` and all internal class references have been updated to reflect the new package structure.
