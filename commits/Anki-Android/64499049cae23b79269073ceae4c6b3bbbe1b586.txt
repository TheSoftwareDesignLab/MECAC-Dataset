Add original default CodeQL workflow generated when switching to Advanced scanning

You can get default codeQL config on Github GUI. Here are steps.
1.On GitHub, navigate to the main page of the repository.
2.Under your repository name, click  Settings. If you cannot see the "Settings" tab, select the  dropdown menu, then click Settings.
3.In the "Security" section of the sidebar, click  Advanced Security.
4.Scroll down to "Code Security", in the "CodeQL analysis" row select Set up , then click Advanced.

please Refer to following URL to get more information about codeQL setup.
https://docs.github.com/en/code-security/code-scanning/enabling-code-scanning/configuring-default-setup-for-code-scanning
