fix(widget): disable "material you" if using MIUI

MIUI doesn't support Material You well - the color is based
on the system background color, with no means to disable it.

https://www.reddit.com/r/Xiaomi/comments/y6zclf/material_you_on_miui_13/

Since we are using a widget, we can't use custom themes as a fix

So to fix the issue: duplicate the assets/layout as unthemed and fix them there
