style(reader): Improve expandable tags chip background theming

Replaces the static `on_surface_divider` color with a new `on_surface_chip`
color resource for the Reader expandable tags view chips.
The new `on_surface_chip` color uses `?attr/colorOnSurface` with a dynamic
alpha value (`expandable_chips_view_chip_alpha`) that is defined separately
for light and dark themes. This ensures better visual consistency and theme
adaptation for the chip backgrounds across the application.
