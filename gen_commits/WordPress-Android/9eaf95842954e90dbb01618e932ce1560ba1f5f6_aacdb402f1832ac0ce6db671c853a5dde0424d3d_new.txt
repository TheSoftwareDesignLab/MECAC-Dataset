feat: Implement new JavaScript-based rich text editor

Introduces a modern JS-powered rich text editor alongside the existing one,
now designated as the legacy editor.

Key changes include:
- Refactored editor architecture to support multiple editor implementations.
- Integrated Rangy.js, js-beautifier, and WordPress content filters for robust HTML editing.
- Added Android-specific JavaScript bridge for native-JS communication and format bar controls.
- Upgraded Android build tools, Gradle plugin, and support library versions.
- Established comprehensive unit and integration testing frameworks.
- Migrated project license from MIT to GPLv2.
- Updated README with build/testing instructions and example app to showcase both editors.
