build: Migrate Navigation Safe Args plugin to plugins block

Migrates the Navigation Safe Args Gradle plugin application from the deprecated
buildscript classpath to the modern plugins block in settings.gradle.

This change includes:
- Centralizing 'navComponentVersion' in settings.gradle.
- Applying the 'androidx.navigation.safeargs.kotlin' plugin.
- Removing the old classpath declaration in the root build.gradle.
- Adding the Google Maven repository to pluginManagement.
- Updating module dependencies to reference the global navComponentVersion.
