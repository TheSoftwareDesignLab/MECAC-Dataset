feat(media): Allow editing of media item captions

Users can now edit the caption for a media item directly from the media edit screen.
The updated caption is saved to the local database and sent to the WordPress server as the 'post_excerpt' field via XML-RPC.

- Removed the "Show Likes" and "Show Sharing Buttons" checkboxes from the media edit layout.
- Cleaned up an unused import in `MediaBrowserActivity`.
