feat(support): Add application logs viewer and sharing

Introduces a new "Application Logs" section accessible from the Support screen.
This feature allows users to view and share application logs, organized by day.

- Implemented using Jetpack Compose for the UI (LogsListScreen, LogDetailScreen).
- Provides navigation to log details and sharing functionality.
- Includes parsing and formatting logic within LogsViewModel.
- Displays the application version name on the main Support screen.
- Adds necessary strings and comprehensive unit tests for LogsViewModel.
