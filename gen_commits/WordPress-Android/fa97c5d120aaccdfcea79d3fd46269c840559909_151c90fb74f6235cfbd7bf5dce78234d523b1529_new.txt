feat: Add subtitle support to WPPrefView text input dialogs

WPPrefView text input dialogs can now display an optional subtitle above the input field.
This is implemented by:
- Introducing a new XML attribute `wpTextDialogSubtitle` for WPPrefView.
- Creating a new layout `wppref_text_dialog.xml` to include a TextView for the subtitle.
- Updating `WPPrefView#showTextDialog` to inflate the new layout and display the subtitle if provided.

This feature is used in `PublicizePrefsFragment` for the "Label" and "Twitter username" preferences to provide additional context.
