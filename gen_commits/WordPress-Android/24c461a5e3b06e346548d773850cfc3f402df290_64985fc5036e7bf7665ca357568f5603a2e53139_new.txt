refactor: Update Aztec library to v1.0-beta.2 and adopt API changes

Updates the Aztec Android library to v1.0-beta.2.
Adopts API renames from `media` to `image` for various methods and listeners, including:
- `OnMediaTappedListener` to `OnImageTappedListener`
- `setOnMediaTappedListener` to `setOnImageTappedListener`
- `insertMedia` to `insertImage`
- `mediaTapped` to `onImageTapped`
