refactor(media): Standardize media item layouts and IDs

Introduces a new `media_picker_file_item` layout for improved reusability.
Refactors existing media item resource IDs in `MediaGridAdapter` and `media_grid_item.xml`
to a more generic naming convention (e.g., `media_item_name`).
Updates `AudioThumbnailViewHolder` to utilize the new common layout.
