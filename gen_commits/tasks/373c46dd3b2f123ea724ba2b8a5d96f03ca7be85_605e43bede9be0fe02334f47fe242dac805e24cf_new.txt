feat: Add configurable row spacing to scrollable widget

Introduce a new `ListPreference` to allow users to set the vertical spacing between task rows in the scrollable widget. Options include default (12dp), compact (6dp), and none (0dp).
This updates the padding of task text, due date, and checkbox elements in the widget rows.
Includes `widget_padding_compact` dimension and refactoring of `Preferences.getIntegerFromString`.
