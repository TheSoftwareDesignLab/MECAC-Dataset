refactor: Consolidate control set click handling

Replaced ButterKnife @OnClick annotations in various ControlSet subclasses
with a new, centralized click handling mechanism.
The base TaskEditControlFragment now provides onRowClick() and isClickable()
methods. Subclasses override these to define click behavior, and the base
fragment sets an OnClickListener on the entire row if isClickable() returns true.
This promotes consistency and standardizes row background to use selectableItemBackground.
