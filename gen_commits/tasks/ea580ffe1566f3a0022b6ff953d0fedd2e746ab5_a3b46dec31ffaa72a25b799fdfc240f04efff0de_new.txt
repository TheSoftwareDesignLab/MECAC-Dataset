feat: Introduce CheckableImageView for task completion control

Refactor task completion checkboxes from Android's native `CheckBox` to a new custom `CheckableImageView`. This provides enhanced flexibility for styling and managing the checked state visually.

Key changes include:
- A new `CheckableImageView` component that extends `ImageView` and implements `Checkable`.
- Replaced `CheckBox` with `CheckableImageView` in layouts: `control_set_title.xml`, `task_adapter_draggable_row.xml`, and `task_adapter_row.xml`.
- Adjusted padding and margins for the completion control in these layouts.
- Updated `TaskAdapter`, `EditTextControlSet`, and `EditTitleControlSet` to utilize the `CheckableImageView` API (e.g., `setImageResource` instead of `setButtonDrawable`, `OnClickListener` instead of `OnCheckedChangeListener`).
