feat: Implement direct text filtering on task list

Introduces the ability to type characters on the task list screen to
dynamically filter tasks by their title.

This commit also includes:
- Refactoring of the backup XML format versioning for clearer separation
  from app version.
- Updates to `TaskService.fetchFiltered` to support the new text
  constraint parameter.
- Adaptations in `TaskAdapter` and `TaskListActivity` to enable and
  manage the new filtering experience.
- Disabled the global debug constant.
