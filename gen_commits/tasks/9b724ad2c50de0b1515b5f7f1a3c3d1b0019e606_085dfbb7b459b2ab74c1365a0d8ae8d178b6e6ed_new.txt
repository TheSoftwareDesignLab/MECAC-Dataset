feat: Create dedicated item type for list actions

Introduces a new 'ACTION' type for FilterListItem to represent clickable actions in lists, such as "Add New List" in the navigation drawer.

This change includes:
- A new 'ActionViewHolder' and 'filter_adapter_action.xml' layout.
- Integration into FilterAdapter and NavigationDrawerAdapter.
- Migration of 'NavigationDrawerAction' to use the new 'ACTION' type.
