feat: Release Astrid 3.9: Redesign, new features, and fixes

This release finalizes version 3.9 (versionCode 202), bringing a refreshed user experience and new capabilities.

Key changes include:
- Implemented a cleaner design, especially on the task edit page, with calendar layout adjustments.
- Added "Beast Mode" for customizing the task edit page.
- Enabled creation of shared lists open to anyone.
- Addressed fixes for some ICS crashes.
- Improved Google Tasks synchronization.
- Minor bug fixes.
- Adjusted A/B testing probabilities for themes to favor the white theme.
