feat: Introduce AndroidX Paging Library for task lists

Integrates the AndroidX Paging Library to improve performance and memory usage for long task lists.

- Refactors `TaskListRecyclerAdapter` into an abstract base class.
- Creates `DragAndDropRecyclerAdapter` to handle manual sorting and subtask reordering.
- Creates `PagedListRecyclerAdapter` for efficient display of non-manually sorted task lists using `PagedList`.
- Updates `TaskListViewModel` to conditionally use `LivePagedListBuilder` when manual sorting is not enabled.
- Upgrades various AndroidX and Kotlin dependencies to their latest stable or release candidate versions.
