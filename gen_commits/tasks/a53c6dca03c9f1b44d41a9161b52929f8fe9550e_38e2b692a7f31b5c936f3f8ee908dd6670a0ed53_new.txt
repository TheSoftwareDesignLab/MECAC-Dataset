chore(release): Publish Astrid 3.0.0

Updated AndroidManifest.xml to versionName="3.0.0" and versionCode="138".
This release includes several bug fixes and improvements:
- Added null check in BackupService to prevent potential NPE when backup directory is undefined.
- Improved TaskListActivity menu handling by clearing old items before populating.
- Prevented TaskListActivity content reload and cleanup on cancelled activity results.
- Enhanced TaskAdapter for more reliable task completion caching and display logic.
- Integrated support for launching addon intents from the task list menu.
