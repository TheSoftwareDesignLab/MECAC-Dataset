feat(ur): Add translations for various UI elements and settings

This commit introduces new Urdu translations for a wide range of application strings.
These include:
- Date and time shortcuts (e.g., "tomorrow morning", "tonight")
- Filter and tag labels
- Location and geofence radius settings
- Reminder and task management actions (e.g., "add location", "remove")
- Various application settings (e.g., quiet hours, notifications, widget configuration, font size, backup options)
