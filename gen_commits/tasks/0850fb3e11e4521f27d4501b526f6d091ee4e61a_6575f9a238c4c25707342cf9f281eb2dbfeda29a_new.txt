refactor: Streamline TaskEditControlSet display views

Removes the redundant `display_row_title` TextView from control set layouts and the corresponding `displayText` field. The `display_row_edit` TextView now exclusively handles the displayed content, aligning left.

Introduces `themeColor` and `unsetColor` in `TaskEditControlSet` to consistently style the displayed text. Empty content states (e.g., untagged, no notes) now use `unsetColor`, while active content uses `themeColor`, improving UI consistency.
