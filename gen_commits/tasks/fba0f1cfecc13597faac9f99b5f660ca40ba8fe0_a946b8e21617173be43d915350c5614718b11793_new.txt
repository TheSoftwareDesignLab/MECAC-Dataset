feat(navigation): Implement collapsible filter and tag groups

Introduced collapse/expand functionality for filter categories in the navigation drawer and remote list pickers.
- Persist collapsed state for Google Tasks and CalDAV accounts in the database.
- Persist collapsed state for built-in filters and tags using preferences.
- Implemented a dedicated `SubheaderViewHolder` to manage UI interaction and state toggling.
- Updated filter providers and adapters to dynamically display sections based on their collapsed state.
- Added new UI elements (up/down arrows) and a broadcast mechanism for real-time UI updates.
- Incremented database version to 72 with migration for new 'collapsed' columns in account tables.
