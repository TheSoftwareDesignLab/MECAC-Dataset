feat: Enable ActionBar and Up navigation across activities

This commit introduces the Android ActionBar (App Bar) for improved navigation and action discoverability.

Key changes include:
- Configure key menu items in `FilterListActivity`, `TaskEditActivity`, and `TaskListActivity` to show in the ActionBar using `setShowAsAction(MenuItem.SHOW_AS_ACTION_IF_ROOM)`.
- Implement "Up" navigation (`android.R.id.home`) handling:
  - `TaskEditActivity`: Saves the current task.
  - `TaskListActivity`: Navigates to `FilterListActivity` when not in dual-pane mode.
  - `FilterListActivity`: Adds a placeholder for potential future dashboard integration.
- Remove `windowNoTitle` from `Theme.Transparent` and `Theme.GreenDroid.NoTitleBar` to allow ActionBar display.
- Adjust `task_edit_wrapper_activity.xml` (landscape layout) to remove the `FilterListActivity` fragment and increase `TaskListActivity` fragment's weight, streamlining the dual-pane layout for task editing.
