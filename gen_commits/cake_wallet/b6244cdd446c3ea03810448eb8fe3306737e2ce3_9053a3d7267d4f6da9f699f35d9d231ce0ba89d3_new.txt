feat: Add Dogecoin wallet support

This commit introduces full support for Dogecoin (DOGE) wallets.
Key changes include:
- Integration of `cw_dogecoin` package for wallet creation, restoration, and management.
- Updates across core components to recognize and handle `WalletType.dogecoin`.
- Implementation of Dogecoin-specific Electrum server lists and node management.
- Addition of Dogecoin URL schemes for Android and iOS.
- UI enhancements for Dogecoin, including icons, transaction details, and address management.
- Updates to build scripts and configuration for Dogecoin integration.
