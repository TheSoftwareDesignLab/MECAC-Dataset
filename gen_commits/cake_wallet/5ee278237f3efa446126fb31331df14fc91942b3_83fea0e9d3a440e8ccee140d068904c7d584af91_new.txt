feat: Add SSL configuration for custom nodes

Introduces a `useSSL` option for custom nodes, allowing users to
specify whether an SSL connection should be used when connecting.

- Persists `useSSL` in the `Node` entity (Hive field 4).
- Adds a 'Use SSL' toggle to the node creation/edit UI.
- Passes the configured `useSSL` value to the Monero wallet connection.
- Implements a new `StandardCheckbox` widget for UI input.
