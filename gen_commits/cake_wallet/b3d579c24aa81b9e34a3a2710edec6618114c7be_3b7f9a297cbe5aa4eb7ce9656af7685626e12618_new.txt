feat(polygon): Add Polygon (MATIC) wallet support

Integrates Polygon (MATIC) blockchain as a new wallet type.
This includes a new `cw_polygon` package with dedicated wallet logic, client, transaction handling, and ERC20 token management.
Extensive updates across the application to support WalletType.polygon, including UI components, deep linking, address validation, fiat conversion, and WalletConnect integration.
Adds PolygonScan API key, default nodes, and privacy settings for PolygonScan history.
Enhances EVM compatibility in the core Ethereum client and WalletConnect services to accommodate Polygon.
