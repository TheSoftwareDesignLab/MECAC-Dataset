refactor(filter): Enhance filter widget responsiveness and layout

Refactors the FilterWidget to use AlertBackground and a flexible AlertCloseButton.
Implements dynamic scrolling for filter sections that exceed screen height, improving UX.
The AlertCloseButton now includes an 'isPositioned' property for versatile placement.
Removes the DropdownFilterItem implementation, streamlining filter types.
Also updates the PR template to include accessibility testing.
