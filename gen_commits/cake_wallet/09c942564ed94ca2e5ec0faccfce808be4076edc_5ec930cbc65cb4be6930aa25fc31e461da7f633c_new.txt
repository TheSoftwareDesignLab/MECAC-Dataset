refactor: Migrate theming to ThemeExtension and add new themes

Introduced a structured theming system using Flutter's ThemeExtension.
Moved UI color definitions and styles into dedicated extensions for improved organization and maintainability.
Added new visual themes: Monero Light/Dark, Bitcoin Light/Dark, Matrix Green Dark, and High Contrast.
Refactored UI widgets to consume theme-specific styles via ThemeExtensions.
Implemented a new `GradientBackground` widget for consistent gradient application.
Updated localization files to include new theme names.
Adjusted Dart SDK constraints in `pubspec.lock` files.
Refactored `SectionDivider` into `HorizontalSectionDivider` and `VerticalSectionDivider`.
