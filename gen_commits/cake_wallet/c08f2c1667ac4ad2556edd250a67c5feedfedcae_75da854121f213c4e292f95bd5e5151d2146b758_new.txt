feat: Allow saving send templates with fiat amounts

This commit introduces the ability to save send templates with amounts specified in either cryptocurrency or fiat currency.

Key changes include:
*   **Template Entity Update**: The `Template` entity now stores `fiatCurrency`, `amountFiat`, and a boolean `isCurrencySelected` to indicate if the amount is in crypto or fiat.
*   **Send Template Page**:
    *   UI updated to allow users to select whether the amount is in crypto or fiat when creating or editing a template.
    *   A new `PrefixCurrencyIcon` widget visually indicates the currently selected currency type for the amount field.
*   **Send Page**: When a template is loaded, the `SendPage` now correctly interprets and pre-fills the amount based on the saved currency type. It also temporarily adjusts the `SettingsViewModel`'s fiat currency if a fiat-based template is loaded, reverting on page close.
*   **XHV Logo**: Added XHV logo for currency display.
