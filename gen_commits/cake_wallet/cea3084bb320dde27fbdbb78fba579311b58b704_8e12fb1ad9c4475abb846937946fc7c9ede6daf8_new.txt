feat: Add Monero UR support for offline transactions & key exchange

This introduces Uniform Resource (UR) support for Monero wallets, enabling offline transaction signing and key image exchange via animated QR codes.

Key changes:
- Implemented `AnimatedURPage` for displaying and scanning multipart UR QR codes.
- Switched to `fast_scanner` for improved QR code scanning capabilities.
- Updated `PendingTransaction` to support UR-based commits, primarily for view-only Monero wallets.
- Integrated Monero API for submitting UR-signed transactions and importing/exporting UR key images/outputs.
- Added primary address display to Monero, Haven, and Wownero wallet keys.
- Enhanced Android CI build with increased Gradle memory and keystore caching.
- Updated default Monero node to be trusted.
