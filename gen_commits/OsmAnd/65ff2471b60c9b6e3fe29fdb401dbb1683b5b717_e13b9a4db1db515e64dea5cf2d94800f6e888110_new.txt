feat: Add rounding option to formatted size values

Introduces a `round` parameter to `AndroidUtils.formatSize` and
`formatRatioOfSizes` methods. When `round` is true, the formatted
size values (GB, MB, kB) will be rounded to the nearest whole number,
providing a cleaner display for metrics.

The MemoryInfoWidget now utilizes this option to display memory
information with rounded values, improving readability.
Additionally, the ratio separator for `formatRatioOfSizes` was updated
for a more compact representation.
