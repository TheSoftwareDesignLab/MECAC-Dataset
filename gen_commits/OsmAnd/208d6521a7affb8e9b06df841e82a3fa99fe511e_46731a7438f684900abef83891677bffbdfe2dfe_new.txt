refactor(osmodroid): Optimize GPX loading and layer access

Refactor OsMoDroid plugin to load GPX data for all layers in a single background task, removing the `layerId` parameter from `getGpxArrayList`. This streamlines the loading process.

OsMoDroidLayer now uses cached references for point lists, avoiding redundant calls to the plugin.

Updated SherlockBar project target API from Android 16 to 17.
