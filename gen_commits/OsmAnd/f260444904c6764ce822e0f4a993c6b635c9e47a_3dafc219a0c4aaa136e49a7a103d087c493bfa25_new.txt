refactor(mcm): Introduce MenuController for object-specific logic

Refactored Map Context Menu to use a MenuController pattern, delegating object-specific logic (icons, names, tap handling, state saving) to dedicated AmenityMenuController and FavouritePointMenuController.
MapContextMenuFragment now supports dynamic content updates via `rebuildMenu()`.
Improved Favorite Point Editor: category selection now uses a dialog instead of direct input and handles empty categories consistently.
Context menu refreshes automatically upon saving an edited favorite.
Single tap on map now allows MenuController to handle specific interactions (e.g., dismissing the favorite editor).
