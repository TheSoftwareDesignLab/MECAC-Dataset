feat: Display bearing for shared live locations

Introduces support for bearing information for live location points.
- Adds `POINT_BEARING_PARAM` to `AMapPoint` for passing bearing data.
- Displays bearing in OsmAnd's map context menu and point details.
- Includes bearing in location text messages sent via Telegram.
- Implements a sending delay for location updates to reduce server load, especially with multiple chats.
- Refines location sharing error handling based on share type (map, text, or both).
- Converts `lastLocationUpdateTime` to seconds for consistency in error checks.
