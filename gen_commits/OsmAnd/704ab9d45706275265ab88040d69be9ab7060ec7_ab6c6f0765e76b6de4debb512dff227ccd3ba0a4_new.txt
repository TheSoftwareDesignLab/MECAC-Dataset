refactor: Migrate custom text input widgets

Replaced `OsmandTextFieldBoxes` and `ExtendedEditText` with new custom widgets in the `net.osmand.plus.widgets` package.
- Deleted `OsmandTextFieldBoxes.java`.
- Introduced `OsmandTextViewFieldBoxes` and `OsmandEditTextFieldBoxes` as new `TextFieldBoxes` implementations.
- Introduced `CursorTextView` as the underlying text input view, allowing for custom cursor drawing.
- Updated `CoordinateInputDialogFragment`, `RenameMarkerBottomSheetDialogFragment`, and `SaveAsTrackBottomSheetDialogFragment` to use the new widgets.

This refactoring centralizes UI components, improving code organization and maintainability.
