i18n: Add extensive new Slovenian translation strings

This commit introduces a large set of new strings for the Slovenian (sl) localization.
It covers various UI components, descriptions, and messages related to:
- GPS points and speed units
- Privacy policy and Telegram integration
- Battery optimization and background work
- Location permissions and sharing controls
- Time settings and formats
- Telegram registration flow
- Error messages
- Disconnection and logout procedures
- Overall UI enhancements.
