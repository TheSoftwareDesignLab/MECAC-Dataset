refactor(wikivoyage): Centralize dialog theme and toolbar setup

Created `WikivoyageBaseDialogFragment` to encapsulate common UI logic for Wikivoyage dialogs.
This includes handling theme selection (light/dark mode) and setting up standard toolbar navigation.
`WikivoyageArticleDialogFragment`, `WikivoyageExploreDialogFragment`, and `WikivoyageSearchDialogFragment`
now extend this base class.
Added new Wikivoyage-specific color resources to support consistent theming across these dialogs.
