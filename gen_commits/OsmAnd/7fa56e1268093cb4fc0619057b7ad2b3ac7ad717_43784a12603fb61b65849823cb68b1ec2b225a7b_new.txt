refactor: Abstract context menu display using ContextMenuAdapter

Replaced direct usage of `net.londatiga.android.QuickAction` with `ContextMenuAdapter`.
The `ContextMenuAdapter` now serves as an abstraction layer, handling both `QuickAction` and `AlertDialog` displays based on accessibility settings.
Introduced `ContextMenuAdapter.setAnchor()` and `MapActivityActions.showObjectContextMenu()` to manage the display logic.
This consolidates context menu creation and display logic across multiple activities.
