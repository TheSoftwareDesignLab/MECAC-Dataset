feat: Add temperature map widget for external sensors

Introduces a new map widget to display temperature data from connected
external sensors, such as BLE devices.

This commit includes:
- New UI strings for the temperature widget.
- Integration of the temperature widget into the External Sensors Plugin.
- Definition of a new `WidgetType.TEMPERATURE`.
- Association of the sensor data field type with the new widget type.
- Addition of a unit (Celsius) for BLE temperature sensor data.
- Configuration to use generic sensor widget settings.
