feat: Improve RTL menu animations and layout support

Adds `slide_in_right` and `slide_out_right` animations for right-to-left (RTL) layouts.
Updates menu controllers (`BaseMenuController`, `MapRouteInfoMenuFragment`) to use these RTL-specific animations in landscape mode.
Enhances RTL compatibility for various UI elements by:
- Adjusting `layout_gravity` to `center_vertical|start`.
- Using `AndroidUtils` for RTL-aware setting of margins, padding, and compound drawables.
- Setting RTL-aware navigation icons for toolbars.
Introduces `right_menu_view_bg` style attribute for right-side menus.
