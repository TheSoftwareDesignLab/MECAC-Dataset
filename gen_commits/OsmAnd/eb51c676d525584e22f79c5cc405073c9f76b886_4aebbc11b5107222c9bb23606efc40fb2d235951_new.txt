feat: Add 'Within polygons' section to map context menu

Introduces a new "Within polygons" row in the map context menu, displaying
administrative boundaries, natural features, or other polygons that contain
the selected map point. Users can tap to view the context menu for these
containing polygons.

- Refactor `MenuObject` creation and text generation into `MenuObjectUtils`.
- Enhance `RenderedObjectMenuController` to improve object name and type
  determination using language, tags, and related POI categories/types.
- Add `MapRendererContext.retrievePolygonsAroundPoint` overload to exclude
  the primary selected object from the "within" list.
