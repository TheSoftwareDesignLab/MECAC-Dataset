feat: Rework profile editing screen UI and logic

Introduces a new UI layout (profile_edit_list_item.xml) for managing application profiles.
The `EditProfilesFragment` and `ProfilesAdapter` are refactored into the `net.osmand.plus.profiles` package
and updated to utilize this new layout. This provides dedicated elements for profile icons,
action buttons (e.g., remove), and a move handle for reordering.

A new string resource `edit_profiles_descr` explains limitations for editing default profiles.
