feat: Introduce customizable map text size setting

This commit adds a new user preference to adjust the size of text labels on the map.
- Implements a 'Text size' list preference in the map settings, allowing users to choose a scaling factor for map labels.
- Integrates the selected text size into the map rendering context.
- Updates the map renderer to apply the chosen scaling factor to all displayed map text.
- Includes refactoring of text drawing logic to better handle text halos/shadows and label intersections.
