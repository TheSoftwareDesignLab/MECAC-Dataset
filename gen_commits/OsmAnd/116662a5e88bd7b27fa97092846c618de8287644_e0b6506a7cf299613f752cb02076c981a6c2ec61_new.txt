feat: Extend AIDL API with customization, voice navigation, and GPX color

This commit introduces several new features and improvements to the AIDL API:

- Add new methods for comprehensive OsmAnd app customization:
  - `setCustomization(in CustomizationInfoParams params)`
  - `areOsmandSettingsCustomized(in OsmandSettingsInfoParams params)`
- Integrate voice navigation message callbacks:
  - `registerForVoiceRouterMessages()`
  - `onVoiceRouterNotify(in OnVoiceNavigationParams params)`
- Introduce `getGpxColor(inout GpxColorParams params)` to retrieve GPX track colors via AIDL, refactoring color handling from `AGpxFile`.
- Add `removeAllActiveMapMarkers()` method.
- Enhance documentation for existing methods: `addContextMenuButtons`, `removeContextMenuButtons`, `updateContextMenuButtons`, `addMapPoint`, `removeMapMarker`, `updateMapMarker`.
- Create new AIDL Parcelable classes for the added functionality.
